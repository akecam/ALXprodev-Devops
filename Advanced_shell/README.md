- Basic knowledge of shell scripting

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/Advanced_shell.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Advanced_shell
   ```

## Usage
Run the shell scripts using the following commands:
```bash
./script1.sh
./script2.sh
```
